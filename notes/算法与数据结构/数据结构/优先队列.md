# Java静态代理与动态代理

我们都知道在 Java 的世界里有静态代理和动态代理，今天就来好好分析下这两种代理方式之间的不同以及其基本用法。

## 代理模式 Proxy

### 意图

 为其他对象提供一种代理以控制对这个对象的访问。

### 别名

Surrogate
